# sg-pd_applications  
A customizable and lightweight **Police Department Application System** for FiveM servers.  
Designed for QBCore (or any framework with minor adjustments), this resource provides an in-game UI for players to submit PD applications, and tools for staff members to review, approve, or deny them.
